# weather-stuff
A Python application that ingests live data from the OpenWeatherMap API using the `requests` library, handles JSON data payloads, and generates trend charts (temperature, humidity, and wind) using `NumPy` and `Matplotlib`. 

# install this library before launching: 
pip install requests matplotlib numpy

incase if the API Key doesnt work anymore, go to https://openweathermap.org/ and make a new free account.  Then, go to "My API Keys", copy the key and paste it in the "API_Key" line.
